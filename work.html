Info

This is a AES encrypted secure web-chat application. I build it for my SDL project it is build in Python, flask, HTML/CSS.

The Challenge

Main challenge was key exchange between two users for AES encryption and decryption of messages.

The Solution

Public and Private key infrastructure is used in this public and private keys are generated using RSA key generation algorithm, then Deffe Hellmen key exchange machanism is implemented to generate a public shared key for which will be diffrent for each and every user. Now a secret key is generated using publicly shared key and private key to encrypt messages with AES algorithm.
